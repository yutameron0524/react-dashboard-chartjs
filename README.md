# 売上管理ダッシュボード Webアプリ

## 概要
売上データを登録・管理し、グラフで可視化できるシンプルな売上管理ダッシュボードです。
売上の推移をグラフで確認することができ、売上データの管理と簡単な分析を行うことができます。

## 使用技術

- React
- TypeScript
- Vite
- Tailwind CSS
- Chart.js

## 機能

- 売上データ登録
- 売上一覧表示
- 売上削除
- 売上グラフ表示（Chart.js）
